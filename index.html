Projects

MoonlightML

An HTML framework written in Lua. It allows for a cleaner syntax than HTML and elegant integration with Lua for procedural creation of page content. This website was made with MoonlightML; each of these project sections are produced with a function.

/RHYTHM

/RHYTHM (read 'slash rhythm') is a mouse motion based rhythm game where you slice notes by moving your cursor. Designed to be community driven, with support for user-made beatmaps using custom songs and even custom mechanics.

Supersonic Shrimpy

A beat-em-up rougelike featuring a mantis shrimp protagonist punching its way through aquatic-themed enemies and enviroments.
